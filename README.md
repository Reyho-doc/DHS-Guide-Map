# DHS-Guide-Map
An interactive web-based map designed to help students, staff, and visitors easily navigate around the school.

This project aims to provide a simple and intuitive way to:
- View school locations by floor
- Search for rooms and facilities
- Get guided routes between locations

The project is being developed as a student-led learning project and is **not an official school system**.

---

## Tech Stack

- **Backend:** Python, FastAPI, SQLite
- **Frontend:** HTML, JavaScript, three.js
- **Pathfinding:** NetworkX
